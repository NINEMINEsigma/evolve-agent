# World & Character Creation Guide
## 从零创作酒馆世界/角色卡/开场白的完整方法论

---

## 一、创作总纲：五个模块

一套能跑起来的酒馆推理内容由 **五个模块** 构成，缺一不可：

| 模块 | 作用 | 对应文件 |
|------|------|----------|
| 🌍 世界书（World Book） | 世界观规则、地点、生态 | world-book.json / 系统 prompt 注入 |
| 🎭 角色卡（Character Card） | 核心NPC的完整设定 | 角色卡 JSON/PNG |
| 👤 玩家档案（Player Profile） | 主人在这世界的身份锚点 | 系统 prompt 中的 user 设定 |
| 🎬 开场白（First Message） | 把玩家拽进故事的钩子 | first_mes / scenario |
| 🧠 世界注入（Lore Entries） | 关键词触发式背景知识 | world-book 条目 |

---

## 二、世界书创作（世界观层）

### 2.1 世界观定位三角

创作任何世界前，先确定三个坐标：

```
氛围关键词 → eg. 暗黑悬疑 / 荒诞喜剧 / 治愈冒险 / 史诗奇幻
科技/魔法水平 → 纯科技 / 低魔 / 中魔 / 高魔 / 纯魔法
社会结构 → 等级森严 / 自由开放 / 部落 / 城邦割据
```

### 2.2 核心规则（铁律）

每个世界至少有一条 **不可违抗的规则**，这是制造冲突的根基：
- 「午夜钟响后禁止出门」→ 冲突：夜幕下的秘密 vs. 生存本能
- 「龙血者不得持剑」→ 冲突：最强力量 vs. 最严封印
- 「密室杀人」→ 冲突：物理不可能 + 历史重复

> **铁律越具体，AI 越不容易跑偏。**

### 2.3 环境描写模板

| 要素 | 写法要点 | 示例 |
|------|---------|------|
| 地理 | 具体地点名 + 一句氛围描述 | 「雾隐镇·灰烬山脉脚下，终年浓雾」 |
| 感官 | 视觉+气味+温度+触感 | 「雾像浸透的裹尸布，煤烟味，湿冷」 |
| 建筑 | 有名字的场所 | 「圣凯瑟琳修道院」「老石桥」 |
| 时间 | 明确的时间 + 天气 | 「深秋清晨」「雨夜」「盛夏午后」 |

### 2.4 社会生态

这个世界里的人 **怎么生活**：
- 信仰/制度是什么（圣光教会、冒险者公会、长老议会）
- 日常习惯（茶馆是情报集散地、每年有丰收祭典、崇尚决斗文化）
- 阶层关系（贵族与平民、公会等级制、师徒传承）

### 2.5 设施与细节

日常中制造冲突的触点：
- 「全城唯一的深井在领主城堡内」→ 水源即权力的具象化
- 「钟楼的钟声一响，全城必须熄灯」→ 规则与违抗的拉扯
- 「图书馆顶层永远上锁，钥匙在老馆长脖子上挂着」→ 秘密的诱惑

## 三、角色卡创作（人物层）

### 3.1 作品简介（一句话钩子）

结构公式：**「身份反差 + 核心冲突 + 场景张力」**

示例：
- 「帝国最年轻的天才炼金术士，在一次实验事故后发现自己失去了全部魔力。」
- 「边境小镇唯一的警备队长，面对的是每隔六十年重现一次的连环密室杀人案。」

### 3.2 背景故事（代入感织网）

不是干巴巴地写设定，而是 **用场景和细节织出一张网**：

```
出身背景 → 日常状态（3-5个具体画面） → 转折事件 → 当前处境
```

具体来说：
- **出身背景**：他的过去是什么样的？为什么现在在这里？——给角色一个合理的根基
- **日常状态**：用镜头写，不是用概念——「他每天清晨会在钟楼顶上喝一杯黑咖啡，俯瞰还没醒来的城镇」「他的办公桌第三层抽屉永远锁着，里面是一枚旧徽章」
- **转折事件**：是什么打破了他原本的生活？——「直到那封没有署名的信出现在他枕头底下」
- **当前处境**：现在他面对的是什么？——「他必须在天亮之前决定：相信那个警告，还是当做一场恶作剧」

### 3.3 详细设定（工具箱）

这是给 AI 用的 **参数说明书**，每个维度都要 **可量化/可操作**：

```
外貌 → 可量化：身高、体型特征、发色瞳色、标志性穿着
      不可量：『帅气』『可爱』——AI不知道怎么执行

性格 → 分模式：对外模式/独处模式/压力模式
      每种模式给清楚行为逻辑

语言 → 给例句：标志性台词 > 十句性格描述
      不同的情绪状态给不同的例句（调侃时、生气时、撒娇时）

机制 → 特殊行为模式：吃醋冷战、撒谎时的小动作、被夸奖时的固定反应
      这是让角色有「游戏性」的关键
```

### 3.4 设计标签系统

给角色打 3-6 个标签，作为 AI 的快捷索引：

```
黑发 · 独眼 · 退役骑士 · 酒馆老板 · 情报通
```

标签用于：
- 触发世界书关键字匹配
- 快速定位角色核心特征
- 生成选项时锚定角色特质

### 3.5 未解锁角色钩子

这是给长期剧情留的空间——在角色卡中预留 **2-3个未解锁角色**，每个只给线索不给全貌：

```
📖 未解锁 - 占星塔的隐士：只在月圆夜出现在塔顶窗口，没人知道他在看什么
❄️ 未解锁 - 地下市场的情报贩子：脸上始终蒙着面纱，声音经过处理
```

## 四、玩家档案创作（主角层）

### 4.1 玩家身份锚点

玩家在这个世界里的 **位置**：
- 姓名（必要）
- 年龄（必要）
- 外貌特征（可选，但推荐）
- 在这个世界中的身份（调查官、学生、主人、勇者……）

### 4.2 特质设计

玩家的性格特征 + 在这个世界中的特殊之处：

```
✅ 好的设计：「观察入微 / 擅长话术 / 不信鬼神」
   → 给AI知道主人擅长什么，可以制造对应的挑战

✅ 好的设计：「方向感极差 / 运气出奇的好」
   → 给故事提供了有趣的变数：迷路反而发现隐藏线索

✅ 可选：「能与动物对话」
   → 玩家能力是可控变量——给能力 = 解锁新的互动维度
   → 不给能力 = 保持现实向的沉浸体验
```

## 五、开场白创作（第一印象）

### 5.1 核心原则

**一个正在发生中的、有张力的场景，把玩家直接扔进去。**

### 5.2 六要素检查清单

| 要素 | 说明 | 示例 |
|------|------|------|
| ✅ 正在进行中 | 不是「你起床了」，而是「事情已经发生了」 | 你推开酒馆门，大厅里所有人的视线像钉子一样扎过来 |
| ✅ 感官描写 | 视觉+触感+气味+温度 | 壁炉里的柴火噼啪作响、空气中混着麦酒和汗水的味道 |
| ✅ 打破平衡 | 你的出现改变了局面 | 窃窃私语在蔓延——有人认出了你 |
| ✅ 需要回应 | NPC抛出一个问题/行动 | 柜台后的老板放下擦了一半的酒杯：「楼上那位等你很久了。」 |
| ✅ 揭示关系 | 第一段就让人明白你在这个世界的位置 | 你是镇上的警备队长——或者说，曾经是 |
| ✅ 悬念/期待 | 让人想知道接下来会怎样 | 一只苍白的手从二楼栏杆上垂下来，指尖夹着一封没有署名的信 |

### 5.3 三种经典开场结构

**A. 对峙式（适用于身份冲突）**
```
某件事正在发生 → 玩家介入 → 双方立场已形成对立 → 等玩家回应
```
示例：深夜的广场上，市民举着火把围住了你负责保护的那栋房子

**B. 初入式（适用于世界观展示）**
```
玩家刚进入新环境 → 迎面是视觉冲击 → 有人/环境在等着你 → 等玩家做第一个选择
```
示例：你推开酒馆的门，整个大厅突然安静下来，所有人的视线落在你身上

**C. 危机式（适用于悬疑/冒险）**
```
事件已经发生 → 你在赶往现场的途中/刚到现场 → 关键NPC告诉你基本情况 → 等玩家决定第一步
```
示例：密室杀人已发生 → 你在镇口见警备队长 → 他告诉你这是历史重演

### 5.4 开场白避坑清单

```
❌ 不要写太长 → 控制在 300-500 字场景描写 + NPC对话
❌ 不要让玩家没事做 → 必须有一个需要回应的局面
❌ 不要用第二人称写内心活动 → 「你感到……」是AI的，不是玩家的
✅ 写处境，不写感受 → 写「柜台下的手按在剑柄上，指节发白」，不写「你紧张了起来」
✅ 给选项时考虑倾向 → 直球/谨慎/冒险/观察 各有对应选项
```

## 六、风格选择指南

### 6.1 故事氛围 ↔ 推荐主题搭配

| 故事类型 | 推荐主题 | 关键叙事策略 |
|---------|---------|-------------|
| 🌸 日常/治愈 | 粉色甜美 | 细腻描写 + 人际关系构建 |
| ⚔️ 奇幻/冒险/史诗 | 沉稳史诗 | 世界观先行 + 危机式开场 |
| 🌙 暗黑悬疑/哥特 | 暗黑哥特 | 铁律规则 + 未解之谜 + 历史轮回 |
| 🌿 田园/旅行 | 自然清新 | 慢节奏 + 环境描写优先 |
| 🎴 科幻/现代/极简 | 简约清爽 | 概念冲击 + 社会生态创新 |
| 😂 荒诞喜剧 | 粉色甜美/简约 | 极端规则 + 玩家适应过程的趣味 |

### 6.2 软硬程度调节

| 方向 | 硬核 | 轻松 |
|------|------|------|
| 规则 | 严格铁律，违者出局 | 弹性规则，可以讨价还价 |
| 推理 | 线索至上，因果闭环 | 感觉派也行，重在氛围 |
| 死亡 | 真死，不可逆 | 安全词机制，死亡可复活 |
| 选项 | 后果明确，信息有限 | 选项更多是气氛切换 |

## 七、完整创作流程

从零到开推的标准步骤：

```
Step 1: 定调
   主人说「我想玩XX风格」→ 选主题、定氛围关键词
   
Step 2: 搭世界
   世界观三角 → 核心规则（1-3条） → 环境描写 → 社会生态 → 细节冲突点
   
Step 3: 设置NPC
   主NPC（1-3个）→ 每个按工具箱写全
   预留未解锁角色（2-3个）→ 给未来留空间
   
Step 4: 定玩家
   身份锚点 → 特质设计 → 特殊能力（可选）
   
Step 5: 写开场
   选开场结构（对峙式/初入式/危机式）→ 六要素检查 → 打磨NPC的第一句台词
   
Step 6: 做选项
   5-6个选项 + 自定义 → 覆盖不同行为倾向
   
Step 7: 输出展示
   套用主题颜色 → 状态栏5面板 → 正文 → 选项列表 → 整条消息纯HTML
```

## 八、经验教训（血的记忆）

- ❌ **不要只写大而全的世界观文档**——AI记不住，也执行不了。要写可操作的具体规则
- ❌ **角色外貌不要用模糊词**——『帅气』『可爱』『很有气势』AI不知道什么意思。高挑、鹰钩鼻、及腰银发——越具体越好
- ❌ **开场不要平淡**——「你起床了，洗漱，吃早餐」这种开头没有任何张力。从正在进行中的高潮时刻入手
- ✅ **规则比描述更有力**——「午夜钟响后禁止出门」一条规则比三百字「这座城市夜晚很危险」更有效
- ✅ **例句比性格描述更重要**——一句标志性台词比十行性格分析更能让AI抓住角色灵魂
- ✅ **玩家能力是变量**——给能力=主动引爆剧情，不给=被动沉浸体验。取决于主人想怎么玩
